## Was ist SDL?

- Simple DirectMedia Layer
- Open source (zlib License)
- Cross-platform (Linux, Windows, Mac, iOS, Android)
- Window manager support (X11, Wayland, ...)
- In C geschrieben
- Bindings für viele Sprachen (C#, Go, Haskell, Lua, OCaml, Python, Rust, Odin, ...)
